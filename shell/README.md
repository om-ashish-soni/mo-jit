# shell — mojit-shell init process

This directory will host `mojit-shell`, the tiny C init process that
runs as pid 1 inside the gated guest.

On startup it:

1. Calls `prctl(PR_SET_DUMPABLE, 0)` and
   `prctl(PR_SET_PDEATHSIG, SIGKILL)` — the guest dies with the host,
   and its memory is opaque to other processes at the same UID.
2. Installs the seccomp-BPF belt-and-braces filter generated by
   `../gate/seccomp.go`. Any socket syscall issued from a code address
   outside the gate's range kills the process with `SIGSYS`.
3. Loads the gum translator, seeds the code cache with the guest ELF's
   entry block, and jumps into translated code.
4. Owns the lifetime of the Go-side gate dispatcher, communicating over
   an in-process C ABI exposed by gum's svc hook.

Landing in **M1** alongside the gum fork. See [../PLAN.md](../PLAN.md).
